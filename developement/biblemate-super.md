# Prompt for Creating biblemate-super skill

```
I like the current biblemate skill.  However, it is kind of rigid with a pre-set of a 6-phase plan, a certain options of tools are already expected to be used for a particular phrase, adopted persona already pre-assigned.  This is good as long as the pre-set framework fit to resolve user requests.  But the single framework may not fit all types of bible-related requests.

Therefore, I want you create an addition skill and a corresponding slash command to it.  It behave like biblemate skill in most areas, e.g. request refinement, study plan generation, multiple persona and skills/tools ochestration, pre-final and final writing.

skill name: biblemate-super

slash command: /biblemate-super

Only exceptions for this additional skill are:

* detailed assessment on user request, refine it, and tailor make bible study plan with multiple phases, without limiting to a fixed framework like in biblemate.  it generates a master study plan dynamically, with multiple phases, choosing best tools for each step under each phase.  it allows not only flexibility, but also using tools best suit for particular step tasks.
* adopt persona as the study goes and choose ones which are the best fit for each step instead of rigid framework
* The current biblemate framework practically does not update plan or supplement plans upon each audit, as it follows a pre-defined framework, but for this newly created skill, set clear goals for each phase.  upon completion of each phase, do series audit.  update plans and run extra steps and skills until the goals for the phase are satisfactory fulfilled, before moving to the next phase.
* learn form the biblemate skill phase 6 about the pre-final and final writing.
* create new persona if needed, e.g. Skill/Phase Output Auditor (choose better names)

Think deep to make this skill best fit all bible-related requests.
```
