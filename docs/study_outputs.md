# Study Outputs Reference Guide

For pastors, theologians, and biblical scholars, having instant access to editable, permanent files is essential. Unlike standard chat assistants where research disappears when the session ends or gets buried in temporary artifact caches, the **Antigravity BibleMate Workspace** automatically writes all theological output directly to physical, editable markdown (`.md`) documents inside your local workspace.

This guide details where those files are saved, how they are named, and how to utilize them in your sermon preparation or academic research.

---

## 📁 Output Directory Structure

All outputs generated by the BibleMate agents and skills are organized under three main directories at the root of your workspace:

```
workspace-root/
├── biblemate/               # All theological analyses, outlines, sermons, and devotions
│   ├── YYYY-MM-DD-HH-MM-SS_desc.md          # Output of individual study skills
│   └── YYYY-MM-DD-HH-MM-SS_study_folder/    # Multi-phase orchestrated studies
│       ├── 000-request_and_study_plan.md
│       ├── 001-bible.md
│       ├── 002-commentary.md
│       └── ...
├── images/                  # All generated biblical illustrations and slides
└── export/                  # Word document exports (.docx) and markdown bundles
    └── docx/
        └── YYYY-MM-DD-HH-MM-SS_document.docx
```

## ✍️ How Outputs Are Saved

The saving behavior depends on whether you run a multi-phase orchestrated study or a single direct slash command/skill.

### 1. Individual Study Skills
When you run individual skills (e.g., `/outline Romans 8`, `/devotion Psalm 23`, `/translate-greek John 1:1`), the agent will formulate the response and automatically save a standalone markdown file directly to the root of the `biblemate/` subdirectory.
*   **Format**: `biblemate/YYYY-MM-DD-HH-MM-SS_description.md`
*   **Example**: `biblemate/2026-06-23-00-15-42_sleep_well_prayer.md`
*   **Benefit**: If you execute the same command multiple times, the unique timestamp prevents files from overwriting each other, allowing you to compare different translations, commentaries, or devotional drafts side-by-side.

### 2. Orchestrated Multi-Phase Studies (`/biblemate` & `/biblemate-super`)
When executing a full multi-phase study, a dedicated timestamped folder is created to group all related research files.
*   **Format**: `biblemate/YYYY-MM-DD-HH-MM-SS_study_title/`
*   **Structure**: Each phase is broken down into structured steps saved sequentially (e.g., `001-bible.md`, `002-original.md`, `006-keywords.md`, `009-pre_final_overview.md`, and `010-final_response.md`).
*   **Benefit**: This maintains a complete, audited paper trail of the exegesis, commentary lookups, theological mapping, and final writing.

## 🛠️ Benefits for Pastoral & Scholarly Workflows

1.  **Fully Editable Text**: All outputs are saved in Standard Markdown (`.md`). You can open them in any text editor, IDE (like Cursor or VS Code), or markdown editor (like Obsidian) to modify outlines, add personal notes, or copy-paste passages directly into presentation slides.
2.  **Version Control (Git Integration)**: Since all files are physical documents in the workspace, you can easily track changes over time or push your entire sermon library to a private GitHub repository. Simply run `/sync` to commit and push all latest outputs automatically.
3.  **Local-First & Offline Access**: Because these files are written directly to your local drive, they are instantly available even when you have no internet access.
4.  **Easy Word Export (`/docx`)**: You can convert any markdown study output to a Microsoft Word document (`.docx`) for printing or editing. Simply run:
    ```
    /docx biblemate/2026-06-23-00-15-42_sleep_well_prayer.md
    ```
    The formatted Word file will be saved in `export/docx/` and can be opened in Microsoft Word or Pages immediately.

## 🖥️ Viewing Saved Files in the Web Application

If you are running the standalone browser-based web application (`web_app.py`), you can browse all these saved files dynamically:
1.  Open the **Left Sidebar Drawer** (using the menu icon at the top left).
2.  Click on any file under **Saved Studies** (`biblemate/`), **Images** (`images/`), or **Word Exports** (`export/`).
3.  The file will open instantly in the **Document Reader** tab, rendering beautiful, readable formatting with full styling.
4.  For Word files, clicking them will trigger an automatic download to your local machine.
