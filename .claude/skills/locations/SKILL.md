# Locations Retrieval and Significance Skill

## Overview
This skill allows the agent to retrieve geographical coordinates, maps, and etymological/historical descriptions of locations mentioned in the Bible, and then conduct a comprehensive study of their historical, geographical, and theological significance.

## Guidelines & Objectives
When executing this skill:
- Always run the python retriever script located at `.claude/skills/locations/locations_query.py` to retrieve location data.
- Execute the script using: `python3 .claude/skills/locations/locations_query.py "<query>"` where `<query>` is the name of the location.
- The script automatically handles fuzzy spelling matching (Levenshtein distance) for locations with minor spelling variations.
- Present the exact output of the script to the user. The script filters out embedded HTML map elements but preserves the Live Google Map link, coordinate details, dictionary descriptions, and occurrences in scripture.
- Keep explanations biblically accurate, structurally clear, and cite specific book, chapter, and verse references for all statements.
- All study outputs generated by the `/locations` command must be saved to the `biblemate/` folder with a timestamp prefix under the Universal Study Output Saving Rule.

## Detailed Exegesis & Significance Analysis
After presenting the retrieved details from the script, expand upon the location's significance by addressing:
1. **Geography & Topography**: Location, physical characteristics, and strategic importance in the ancient world.
2. **Historical Timeline**: Major events associated with this place throughout biblical and secular history.
3. **Theological Significance**: Covenants, prophecies, or symbols connected to the location.
4. **Redemptive Fit**: How this place fits into the geography of salvation history (e.g., wilderness, Jerusalem, Babylon).
- If a singular name is used to denote various different locations in the Bible, kindly provide separate descriptions and analysis for each one.
